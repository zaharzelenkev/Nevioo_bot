# NEVIO — ИИ-помощник для подготовки к ЦТ/ЦЭ Беларуси

> Умный Telegram Mini App + Web-приложение для подготовки к **ЦТ** (централизованному тестированию после 11 класса) и **ЦЭ** (централизованному экзамену после 9 и 11 классов) в Беларуси.
> На 100% бесплатно. На базе Groq AI (Llama 3.3 70B).

---

## ⚠️ ВАЖНО: БЕЗОПАСНОСТЬ

Твой старый код содержал **API-ключ Groq в открытом виде** в файле `index.html`. Этот ключ был опубликован в публичном GitHub репозитории, что означает:

- Любой пользователь мог скопировать ключ и использовать за твой счёт
- Groq мог заблокировать твой аккаунт (нарушение TOS)
- Квоты могли улететь за минуты

**Что сделано:**
1. ✅ API-ключ удалён из клиентского кода
2. ✅ Все запросы идут через серверный прокси `/api/*` на Vercel Functions
3. ✅ Ключ хранится в Environment Variables (не виден никому)

**Что нужно сделать ТЕБЕ:**
1. 🔴 **СРОЧНО** зайди в https://console.groq.com → API Keys → **Revoke** старый ключ
2. 🔴 Создай **новый** ключ
3. 🔴 Добавь его на Vercel: Settings → Environment Variables → `GROQ_API_KEY`

## 🚀 Деплой на Vercel

### Шаг 1: Подготовка

```bash
# Скопируй файлы в свой репозиторий:
git init
git add .
git commit -m "NEVIO with secure AI backend"
git remote add origin https://github.com/zaharzelenkev/nevio.git
git push -u origin main
```

### Шаг 2: Деплой

1. Зайди на https://vercel.com → **Add New Project**
2. Импортируй репозиторий `nevio`
3. Настройки:
   - Framework Preset: **Other**
   - Build Command: (пусто)
   - Output Directory: `.`
4. **Environment Variables** → добавь:
   - `GROQ_API_KEY` = `gsk_...` (твой новый ключ)
5. Нажми **Deploy**

### Шаг 3: Готово!

Сайт будет доступен по адресу типа `https://nevio.vercel.app`

## 🧠 API Endpoints

| Endpoint | Что делает |
|----------|------------|
| `/api/chat` | Основной чат со стримингом |
| `/api/test-generator` | Генерация тестов ЦТ/ЦЭ |
| `/api/explain` | Объяснение тем, разбор заданий |
| `/api/vision` | Разбор заданий по фото (Llama Vision) |
| `/api/groq` | Универсальный прокси для Groq |

Все endpoints автоматически:
- Скрывают API-ключ от клиента
- Ограничивают rate-limit (30 запросов/мин на IP)
- Возвращают понятные ошибки на русском

## 💰 Стоимость: БЕСПЛАТНО

Используется Groq Free Tier:
- **Llama 3.3 70B Versatile** — основная модель (для текста)
- **Llama 3.2 90B Vision Preview** — для разбора фото
- Бесплатные лимиты Groq: ~14 400 запросов/день
- Vercel Hobby Plan: 100 GB трафика/мес, serverless функции бесплатно

**При текущей посещаемости (даже 1000+ учеников) проект будет полностью бесплатным.**

## 🛡️ Безопасность

- ✅ API-ключ Groq хранится только на сервере (Vercel Environment Variables)
- ✅ Rate-limit: 30 запросов/мин на IP
- ✅ CORS настроен
- ✅ Никаких пользовательских данных не сохраняется (только localStorage в браузере)
